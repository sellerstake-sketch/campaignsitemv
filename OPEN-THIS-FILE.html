You're opening this file directly from your computer.

The application MUST be accessed through a web server!

Click Here to Open Application (Port 8000)
Click Here to Open Application (Port 8080)

How to Access the Application:

  1. Make sure the server is running (check terminal)
  2. Open your web browser
  3. Type in the address bar: http://localhost:8000/index.html
  4. DO NOT double-click the HTML file
  5. DO NOT use file:// paths

If Server is Not Running:

  1. Open Command Prompt or Terminal
  2. Navigate to this folder: cd C:\Users\USER\Desktop\4.0
  3. Run: python -m http.server 8000
  4. Then open: http://localhost:8000/index.html
